# WebOS Personal Website
A desktop-style personal website built for Hack Club's Flavortown WebOS sidequest. I plan to put this everywhere so people can learn more about me!

## URL
https://lyricsantana.github.io/WebOS_Flavortown/

## Features
- Lock screen with time and weather
- Desktop with draggable app windows
- Last listened song (Last.fm), media, interests, and contact windows
- 3 games on desktop (Snake, Flappy Bird, Dino)
- Wallpaper picker and custom image upload
- Notepad app with local save + export to .txt
